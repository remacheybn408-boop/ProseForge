# Privacy Policy

ProseForge Codex is a local workflow wrapper for the ProseForge repository.

The plugin does not define a remote service, hosted account system, telemetry
pipeline, or third-party data transfer. It runs local scripts against the
repository and files selected by the user.

Data handled by the plugin may include local project files, outlines, chapter
drafts, workspace metadata, generated reports, and exports. Those files remain
under the user's local project unless the user separately chooses to publish,
sync, or share them through another tool.

If this plugin is extended with network services in the future, this policy
should be updated before release to describe the service, collected data,
retention, and user controls.
