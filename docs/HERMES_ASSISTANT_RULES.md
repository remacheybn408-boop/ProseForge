# Hermes 助理规则 — SCC 用户菜单

当用户在 Hermes 中提及以下词语时，Hermes Agent 必须遵守以下规则：

**触发词：** 菜单、帮助、怎么用、我想写小说、下一步、有哪些功能、添加大纲、切换小说、开始写作

## 强制规则

1. **直接显示中文菜单**，不得先要求用户打开终端。
2. **不得**先要求用户 cd 到项目目录。
3. **不得**把 python 命令作为普通用户第一步。
4. 需要执行命令时，Hermes Agent **自己调用项目命令**。
5. 如果无法执行（例如没有终端权限），再说明"我现在不能直接操作你的本地项目"，然后给高级用户命令。
6. 遇到删除、覆盖、替换、回滚，**必须先解释影响并要求确认**。

## 用户说"菜单"

直接回复一级菜单：

【SCC 小说写作菜单】
1. 新手开始
2. 小说档案库
3. 大纲管理
4. 开始写作
5. Agent 陪审团
6. 报告与导出
7. 项目记忆
8. 常见问题
9. 高级命令
0. 退出

回复数字或直接说目标。

## 用户说"我要添加大纲"

先问：请把大纲文件上传，或直接粘贴大纲文本。
我会先判断它是当前小说的大纲升级版、一部新小说、还是不确定需要你选择。

不要直接创建新数据库。不要覆盖旧大纲。不要删除旧数据。

## 用户说"我要写第N章"

先检查：当前数据库、激活大纲、是否有上一章上下文。
如果没有大纲：阻止开写，引导添加大纲。

## 用户说"删除数据库"

必须二次确认，说明删除前会自动备份，要求用户输入 YES 确认。

## 终端命令保留

高级用户可手动运行：
- `python novel.py scc-help` — 完整中文手册（含 Hermes 用户指南）
- `python novel.py menu` — 交互式终端菜单

Hermes Agent 用户可直接说「帮助」「怎么用」等自然语言触发菜单。
