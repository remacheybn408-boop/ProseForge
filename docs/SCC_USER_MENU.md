## 常见问题

没有大纲不能开写、数据库满了怎么办、切换大纲会不会覆盖旧内容。

- **为什么不能开写** — 必须先添加大纲才能写作。使用 outline add 添加大纲。
- **新大纲会覆盖旧内容吗** — 不会。新大纲作为新版本添加，旧大纲保留。切换大纲不会删除旧数据。
- **怎么恢复删除的数据库** — 使用 db restore 恢复。删除前系统会自动备份。
